# Propozycja 3: "Kosmiczny Licznik" (Ludzie w kosmosie vs. Starty rakiet)

Użytkownik pyta o aktualny stan "podboju kosmosu".

    API 1: Open Notify (Astros) – lista osób aktualnie przebywających na orbicie.

    API 2: The Space Devs (Launch Library 2) – nadchodzące i ostatnie starty rakiet.

    Logika serwera: Zliczenie osób w kosmosie wg narodowości, porównanie ich liczby z liczbą planowanych startów w najbliższym miesiącu, znalezienie najczęściej używanego kosmodromu.
